# Self Improvement

> **로드 시점**: 스킬을 스스로 개선되게 / PROMOTE / skill-enhancer 강화 시.

Use real failures as the input. Do not improve a skill because it could hypothetically do more.

## Loop

1. Capture the exact confusing prompt, audit finding, or failed command.
2. Identify the smallest missing instruction, reference, or script behavior.
3. Patch that one thing.
4. Re-run the smallest check that would have caught it.

Promote a pattern only after it appears in more than one skill or prevents a repeated mistake.
